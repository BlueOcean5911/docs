### Create a new company

1. Go to the Companies page.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/company-create-001.png"
  alt="Company Create Light"
/>

2. Click on the "Create Comapny" button. In this stage, you can choose the method to create a new company. There are two options: Import single and Import from CSV.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/company-create-002.png"
  alt="Company Create Light"
/>

- Import single: This option allows you to create a single company by filling in the required fields.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/company-create-003.png"
  alt="Company Create Light"
/>

- Import CSV: This option allows you to import multiple companys from a CSV file.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/company-create-004-csv.png"
  alt="Company Create Light"
/>

You can download the CSV template by clicking on the "Download sample Template" button.
