### Create a new lead

1. Go to the Leads page.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/lead-create-001.png"
  alt="Lead Create Light"
/>

2. Click on the "Create Lead" button. In this stage, you can choose the method to create a new lead. There are two options: Import single and Import from CSV.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/lead-create-002.png"
  alt="Lead Create Light"
/>

- Import single: This option allows you to create a single lead by filling in the required fields.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/lead-create-003.png"
  alt="Lead Create Light"
/>

- Import CSV: This option allows you to import multiple leads from a CSV file.

<img
  className="ml-16 block rounded-2xl shadow-lg"
  src="/images/lead-create-004-csv.png"
  alt="Lead Create Light"
/>

You can download the CSV template by clicking on the "Download sample Template" button.

### Update a lead

   <img
     className="ml-16 block rounded-2xl shadow-lg"
     src="/images/lead-update-001.png"
     alt="Lead Update Light"
   />
2. Save the updated lead information by clicking "Update" button.
   <img
     className="ml-16 block rounded-2xl shadow-lg"
     src="/images/lead-update-002.png"
     alt="Lead Update Light"
   />
