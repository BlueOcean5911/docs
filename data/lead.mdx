### Create a new lead

1. Go to the Leads page.

<img
  className="ml-16 block rounded-2xl shadow-lg dark:hidden"
  src="/images/lead-create-001.png"
  alt="Lead Create Light"
/>

2. Click on the "Create Lead" button. In this stage, you can choose the method to create a new lead. There are two options: Import single and Import from CSV.

<img
  className="ml-16 block rounded-2xl shadow-lg dark:hidden"
  src="/images/lead-create-002.png"
  alt="Lead Create Light"
/>

- Import single: This option allows you to create a single lead by filling in the required fields.

<img
  className="ml-16 block rounded-2xl shadow-lg dark:hidden"
  src="/images/lead-create-003.png"
  alt="Lead Create Light"
/>
